# 狩猟鳥獣診断 テストケース一覧

各動物が必ず診断結果として選ばれる設問回答パターン（12問分）を記載。

---

## テストケースのルール
- 各動物の属性（personality, environment, relation）を4問ずつ選択肢として並べる。
- 例：personalityがA、environmentが2、relationがβの場合
  - 回答例: [A, A, A, A, 2, 2, 2, 2, β, β, β, β]
- これにより、その動物が必ず診断される。
